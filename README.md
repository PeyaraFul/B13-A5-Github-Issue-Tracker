******** ANSWER*********
1. var: It's version of before ES6. Same variable can be redeclare and it can be used before declaration.
let: It can't access outside of blockscope. Variable can't redeclare but it's value can be changed.
const: It follow blockscope and it's value can't be changed.

2. To catch or access every item of array and object we use spread operator(...) . We can also copy a array or object.

3. map(): It's works on every element of a array and return a new array.
filter(): It's checked a specific condition in a array and return a new array.
forEach(): It's works on every element of a array like a loop but doesn't return anything.

4. arrow function: It's updated version(ES6) of function(). By using => sign it is declared also shorted and readable.

5. template literal: It is most powerful way to write string and variable, allow multiline string  writing. In the backticks sign ` ` all the code is written. To show API data and to make DOM element commonly we use template literal `` .
